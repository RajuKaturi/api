Routable

  • Routable(options?: IRoutableClassOptions): any
    • Defined in core/@routable/routable.ts:129

    Decorator applied to classes that represent routing logic for SakuraApi.

    Example

    @Routable({
   baseUrl: 'users'
})
class User {
   @Route({
     path: ':id'.
     method: 'get'
   })
   getUser(req, res) {
     res.status(200).json(someModel.toJson());
   }
}

    The above example creates a routable class called User that has one route, users/:id, which responds to GET requests.

    Automagical Behavior

    Keep in mind that @Routable will instantiate the class and pass it to SakuraApi.route, unless you set the [[RoutableClassOptions.autoRoute]] to false.

    Parameters

    • Optional options: IRoutableClassOptions

    Returns any

Route

  • Route(options?: IRoutableMethodOptions): (Anonymous function)
    • Defined in core/@routable/routable.ts:202

    Decorator applied to methods within an @Routable decorated class that designates that method as a route handler.

    By default, a route that isn't provided a [[RoutableMethodOptions.method]] option, will default to GET. The [[RoutableMethodOptions.path]] defaults to ''.

    See Routable for an example of how to use @Route.
